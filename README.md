Combino uma forte base no mercado imobiliário com habilidades avançadas em desenvolvimento de software para construir aplicações web modernas, sistemas de automação e experiências digitais interativas.

# 🏢 Projetos em Destaque

## 1. Plataforma Imobiliária JAMStack (Augusto Narloch Imóveis)
Plataforma imobiliária de alta performance desenvolvida com geração estática, arquitetura headless CMS e um pipeline de processamento dinâmico de imagens.
- **Destaques Técnicos:** SSG, injeção dinâmica de marca d'água via URL (`fl_layer_apply`), galerias interativas sem distorção (`PhotoSwipe`) e consultas customizadas via GROQ.
- **Stack:** Astro, Sanity CMS, Cloudinary, Vercel.
- 🔗 **Site em Produção:** [augusto-narloch-imoveis.vercel.app](https://augusto-narloch-imoveis.vercel.app/)
- 🔗 **Repositório:** [github.com/augustonarlochcordeiro/augusto-imoveis-site](https://github.com/augustonarlochcordeiro/augusto-imoveis-site)

## 2. TikTok Crisis - Sistema Interativo FiveM
Sistema procedural avançado para servidores FiveM que integra eventos de transmissões ao vivo (lives) diretamente nas ações e eventos do jogo.
- **Destaques Técnicos:** Game scripting, pontes com APIs externas, geração dinâmica de missões (Siege/Convoy).
- **Stack:** Lua, Python, API do FiveM.
- *(Repositório Privado / Em Desenvolvimento)*
