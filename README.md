# 🚀 Sobre Mim

Sou um Engenheiro de Software com formação em Automação Industrial, focado na construção de ecossistemas tecnológicos completos — do hardware à nuvem, do back-end às interfaces web dinâmicas.

Minha transição da automação para o desenvolvimento de software me proporcionou uma visão arquitetural sólida. Não escrevo apenas código; eu projeto sistemas escaláveis, garantindo que a infraestrutura, a modelagem de dados e a experiência do usuário funcionem em perfeita sincronia. 

Busco atuar na resolução de problemas complexos, desenvolvendo APIs robustas, arquiteturas web de alta performance e integrações com dispositivos inteligentes e sistemas distribuídos.

---

# 🎯 Áreas de Expertise & Foco

- **Arquitetura & Full-Stack:** Desenvolvimento de aplicações web modernas (SSR/SSG), APIs RESTful e modelagem de arquiteturas desacopladas (Headless CMS).
- **Performance Web:** Otimização agressiva de Core Web Vitals, tratamento dinâmico de mídia na nuvem e entrega via CDN.
- **Sistemas Distribuídos & Nuvem:** Estruturação de microsserviços, Serverless e gerenciamento de infraestrutura escalável.
- **IoT & Automação:** Integração hardware-software utilizando telemetria avançada, protocolos assíncronos (MQTT) e microcontroladores.
- **Lógica Procedural:** Desenvolvimento de scripts complexos para back-ends e pontes para eventos assíncronos em tempo real.
